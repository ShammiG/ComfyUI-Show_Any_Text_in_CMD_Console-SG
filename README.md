# ComfyUI Show Any Text in CMD Console-SG        

Simple node to show ANY output text in the CMD console with color options.
<br>

<img width="847" height="963" alt="show any text in CMD console comfyUI" src="https://github.com/user-attachments/assets/e4ccf503-7c1a-4706-9116-fd92125b60bc" />
<br>
<br>

● Optional colored text with Hex code or quick list.     
     
You can connect this node to any text output and it will show up on the console.          
           
● Toggle to show/hide text on the node          
            
Works with Nodes 2.0                      

Made this node before [ComfyUI Text_Tools_SG](https://github.com/ShammiG/ComfyUI_Text_Tools_SG)   
thought someone might find this useful

# Installation:   
<br>

**OPTION 1 :** If you have [ComfyUI-Manager](https://github.com/Comfy-Org/ComfyUI-Manager):       
        
**1.** Click on Manager>Custom Nodes Manager           
            
**2.** you can directly search Show_Any_Text_in_CMD_Console-SG **OR** my Username: ***ShammiG*** and click install.           
              
**3.** Restart comfyUI from manager:     
<br> 

**OPTION 2 :** If you don't have comfyUI Manager installed:           
          
**1.** Open command prompt inside ComfyUI/custom_nodes directory.              
       
**2.** Clone this repository into your **ComfyUI/custom_nodes** directory:    
       
    git clone https://github.com/ShammiG/ComfyUI-Show_Any_Text_in_CMD_Console-SG.git
      
**3.** **Restart ComfyUI**             
  Search and add the node to your workflow.
